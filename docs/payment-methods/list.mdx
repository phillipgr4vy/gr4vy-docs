To list all the vaulted payment methods stored in Gr4vy, call the
[`GET /payment-methods`][api] API.

<Tabs
  groupId="language"
  defaultValue="curl"
  values={[
    { label: 'cURL', value: 'curl', },
    { label: 'Node', value: 'node', },
  ]}
>
  <TabItem value="curl">

```bash
curl -i -X GET "https://api.{gr4vyId}.gr4vy.app/payment-methods" \
    -H "Authorization: Bearer [JWT_TOKEN]"
```

  </TabItem>
  <TabItem value="node">

```js
const paymentMethods = await client.listPaymentMethods();
```

  </TabItem>
</Tabs>
